# X 大学 紙容器移行 ケーススタディ

X 大学のカフェテリアは 2019 年にプラスチック容器を全面廃止し、紙容器へ移行した。導入初年度は容器コストが約 3 倍に膨らみ、学食メニューの一部に値上げ圧力がかかった。

しかし 2 年目には仕入れルートの最適化と紙容器メーカーとの長期契約締結により、容器コストは従来比 1.2 倍まで低下した。3 年目には学食価格を導入前水準に戻し、紙容器分のコスト増は他のオペレーション効率化で吸収できた。

この事例から、紙容器への移行に伴うコスト増は短期的なものであり、長期的には持続可能な選択肢であると言える。学生からのアンケートでも、3 年目の段階で紙容器への切り替えに対する満足度が 78% に達している。
