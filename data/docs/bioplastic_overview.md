# バイオプラスチック 概要

バイオプラスチックは、バイオマス由来の原料から作られたプラスチックの総称である。代表的な素材として PLA (ポリ乳酸) と PBS (ポリブチレンサクシネート) があり、いずれも特定の条件下で生分解する。

最新世代のバイオプラ容器は、従来の石油由来プラスチックと比べて約 +40% のコストで製造可能になっており、コストギャップは 5 年前と比べて大幅に縮小している。一方で、生分解には高温堆肥化施設が必要であり、家庭用ゴミとして処理した場合は通常のプラスチックと同様に環境負荷を持つ可能性がある。

つまり、バイオプラスチックは「廃止」と「現状維持」の中間にある第 3 の選択肢として位置づけることができる。ただし、回収・処理の仕組みが整っていない環境ではメリットを十分に発揮できない。
