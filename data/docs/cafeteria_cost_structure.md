# カフェテリア コスト構造の基礎

学食メニュー 1 食あたりのコスト構成は、食材費が約 50%、人件費が約 25%、光熱費・什器費が約 15%、容器・備品費が約 10% である (大学生協連合, 2022)。

容器費が 3 倍に増加した場合、メニュー単価への影響は理論上 +20% となる。ただし、これは他項目を一切調整しない場合のシナリオであり、実際には食材調達の最適化、メニュー構成の見直し、提供量調整などで部分的に吸収できる。

過去事例では、紙容器への移行直後に値上げを実施した大学は 3 校中 2 校で、いずれも 2 年以内に元の価格水準まで戻している。「コスト増 = 永続的な値上げ」という単純な対応関係は成立しないと言える。
